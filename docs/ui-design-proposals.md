
# הצעות עיצוב לאפליקציית ניהול לו״ז מרובת משתמשים

## הנחות יסוד (משותף לכל ההצעות)
- יש שני סוגי משתמשים: **משתמש רגיל** ו-**אדמין**.
- רק **אדמין** יכול לעדכן את סטטוס הלו״ז (למשל: טיוטה / מאושר / נדחה / פורסם).
- משתמש רגיל יכול: להתחבר, לצפות בלו״זים, ליצור לו״ז חדש, לצפות בפרטי לו״ז, ולעדכן הגדרות אישיות.
- סטטוס לו״ז מוצג לכל המשתמשים לקריאה בלבד, אך פעולת השינוי זמינה רק במסך ניהול סטטוס לאדמין.

### טיפוגרפיה
- כותרות: **Heebo Bold**
- טקסט גוף: **Assistant Regular**
- גדלים מומלצים:
  - H1: 32px
  - H2: 24px
  - H3: 20px
  - Body: 16px
  - Caption: 14px

### רכיבי UI
- כפתורים מרובעים עם פינות 8px
- טבלאות עם zebra rows לרשימת לו״זים
- Status Chips בצבעים לפי סטטוס
- Modal לאישור פעולות אדמין (שינוי סטטוס)
- Form fields עם תווית מעל השדה ו-helper text

### ניווט
- Desktop: סיידבר שמאלי קבוע
- Mobile: Bottom Navigation עם 4 אייקונים עיקריים
- Breadcrumbs במסכי עומק (פרטי לו״ז / יצירה)

### היררכיית מידע
1. מידע קריטי: סטטוס נוכחי + תאריך עדכון אחרון
2. פרטי לו״ז מרכזיים (שם, טווח תאריכים, בעלים)
3. תוכן מפורט/הערות
4. היסטוריית שינויים (לאדמין בולט יותר)

### תכנון מסכים
- **התחברות**: טופס נקי במרכז, זכור אותי, שכחתי סיסמה.
- **רשימת לו״זים**: טבלה עם סינון לפי סטטוס, חיפוש לפי שם, מיון לפי תאריך.
- **יצירת לו״ז**: Wizard בן 2 שלבים (פרטים בסיסיים → תצוגה מקדימה ושמירה).
- **פרטי לו״ז**: כרטיס עליון עם סיכום, לשוניות: פרטים / הערות / היסטוריה.
- **ניהול סטטוס לאדמין**: מסך ייעודי עם תור בקשות ועדכון סטטוס ב-bulk.
- **הגדרות משתמש**: פרופיל, שינוי סיסמה, העדפות התראות.

  - H1: 34px
  - H2: 26px
  - H3: 20px
  - Body: 16px
  - Small: 13px

### רכיבי UI
- Cards עם צל רך ופינות 14px
- כפתורי Primary עם gradient עדין
- טפסים עם אייקון בתוך השדה
- Timeline ויזואלי במסך פרטי לו״ז
- Toast notifications לפעולות שמירה/עדכון

### ניווט
- Top Nav עם quick actions
- Sub-navigation בלשוניות בתוך מסכים מורכבים
- Floating Action Button ל"יצירת לו״ז" במובייל

### היררכיית מידע
1. KPI cards בראש רשימת לו״זים (סה"כ/ממתינים/מאושרים)
2. רשימת פריטים עם אינדיקציה חזותית לסטטוס
3. פירוט לפי צורך (expand/collapse)
4. פעולות משניות מוסתרות בתפריט "עוד"

### תכנון מסכים
- **התחברות**: Split layout (צד אחד טופס, צד שני אילוסטרציה).
- **רשימת לו״זים**: Grid cards + פילטרים אופקיים מהירים.
- **יצירת לו״ז**: Form דינמי עם בדיקות בזמן אמת והדגשת שגיאות.
- **פרטי לו״ז**: Header summary + timeline + קבצים מצורפים.
- **ניהול סטטוס לאדמין**: Queue board (דמוי Kanban) לגרירת פריטים בין סטטוסים (רק אדמין).
- **הגדרות משתמש**: קטגוריות בכרטיסיות: חשבון, התראות, שפה וזמינות.

### טיפוגרפיה
- כותרות: **Inter Bold**
- טקסט גוף: **Inter Regular**
- נתונים בטבלאות: **Roboto Mono** (אופציונלי)
- גדלים:
  - H1: 30px
  - H2: 22px
  - Body: 15px
  - Table text: 14px

### רכיבי UI
- Data Table מתקדמת (עמודות ניתנות להתאמה אישית)
- פילטרים מרובי תנאים + שמירת תצוגה (Saved Views)
- Drawer צדדי לעריכת פרטי לו״ז מהירה
- Badge role ברור (Admin/User)
- Audit log מובנה בפרטי לו״ז

### ניווט
- סיידבר + command bar עליון
- קיצורי מקלדת (למשל `/` לחיפוש)
- ניווט מהיר בין לו״זים סמוכים מתוך פרטי הפריט

### היררכיית מידע
1. סרגל בקרה עליון: חיפוש, פילטרים, פעולות bulk
2. טבלת נתונים עם צפיפות גבוהה
3. פאנל פרטים צדדי
4. Audit trail והיסטוריה מלאה

### תכנון מסכים
- **התחברות**: טופס קומפקטי, דגש על אבטחה (2FA אופציונלי).
- **רשימת לו״זים**: טבלת-על עם סינון מתקדם, bulk select ו-export.
- **יצירת לו״ז**: תבניות מוכנות מראש + שדות חובה מסומנים בבירור.
- **פרטי לו״ז**: Layout דו-עמודתי (תוכן + audit).
- **ניהול סטטוס לאדמין**: מסך batch operations עם אישור כפול לפעולות המוניות.
- **הגדרות משתמש**: הרשאות, אזור זמן, פורמט תאריך, נראות עמודות.

- Bottom sheet לפעולות מהירות
- כרטיסי לו״ז עם מידע חיוני בלבד
- מתגים (Switches) פשוטים בהגדרות משתמש

### ניווט
- Bottom Tab Bar: בית / לו״זים / יצירה / הגדרות
- App Bar עם חיפוש גלובלי
- מחוות החלקה (Swipe) לפעולות מהירות על פריט

### היררכיית מידע
1. מידע קצר וברור מעל לקפל (Above the fold)
2. CTA יחיד ראשי בכל מסך
3. פעולות מתקדמות מאחורי "אפשרויות נוספות"
4. תוכן מפורט נפתח באופן מדורג (Accordion)

### תכנון מסכים
- **התחברות**: מסך אנכי מינימליסטי עם שדה טלפון/אימייל וסיסמה.
- **רשימת לו״זים**: כרטיסים אנכיים + פילטר צף מהיר.
- **יצירת לו״ז**: Step-by-step קצר עם autosave.
- **פרטי לו״ז**: Hero status בחלק העליון + פירוט מתקפל.
- **ניהול סטטוס לאדמין**: מצב "Admin Mode" עם בקרים גדולים ואימות נוסף לפני שינוי.
- **הגדרות משתמש**: רשימה פשוטה של העדפות עם חיפוש בתוך הגדרות.
